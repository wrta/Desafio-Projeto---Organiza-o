# Anotações Relevantes
## Passos básicos para adicionar um repositório ao git
### 1 - Criar um novo repositório no git hub
### 2 - Clonar o repositório para um diretório local
#### 2.1 - Git Bash Here no diretório desejado, git clone (chave ssh), git status ao final
### 3 - Após realizar os trabalhos na máquina local
#### 3.1 - git add . // para adicionar todos os arquivos para a lista a ser commitada
### 4 - git commit -m // para criar um commit com as alterações do repositório  (-m é para inserir uma mensagem no commit)
### 5 - git push origin main // para commitar as alterações, ou seja, enviar os arquivos para o repositório em nuvem
